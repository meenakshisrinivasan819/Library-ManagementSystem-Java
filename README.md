# 📚 Library Management System (Java)

## 📌 Overview

Library Management System is a console-based Java application that helps manage books in a library. The system allows users to add, search, issue, return, view, and delete books while maintaining book availability status.

This project demonstrates Java programming fundamentals, Object-Oriented Programming (OOP), and the Collections Framework.

---

## ✨ Features

* ➕ Add New Books
* 📖 View All Books
* 🔍 Search Book by ID
* 📤 Issue Books
* 📥 Return Books
* ❌ Delete Books
* 📊 Library Statistics Dashboard
* ✅ Book Availability Tracking
* 🔄 Real-Time Status Updates
